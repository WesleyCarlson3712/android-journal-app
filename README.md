# Overview
This is a journal app I made as a 2 week project. It is the first mobile app I have ever made, so I wanted to start off with something small and simple. Upon opening the app, you will be able to press the + button to create a journal entry. Once you have created an entry, it will appear on the home screen. That's about all there is to it. A very simple journal app where you can create and view a list of entries sorted by date.

I started this project so I could learn more about app developenemt and how it works. I am happy with how it turned out and may continue adding features in the future.

[Software Demo Video](https://youtu.be/RhvBY1phMzs)

# Development Environment

This app was made entirely in android studio using kotlin. I have never used Android studio or kotlin before so learning both at once while creating an app was a challenge.
I used ChatGPT a fair bit to teach me how apps are meant to be structured in android studio, and to show me the kotlin syntax I would need to create my ideas.

# Installing the App
1. Download JournalApp.apk
2. Open the file on your Android phone
3. If prompted, allow installs from unknown sources
4. Tap Install
5. Open the app
